## Getting started 


### Clone Repo
[https://github.com/phanki5/web_app_anime.git]

### Clone repo

```bash

git clone https://github.com/phanki5/web_app_anime.git

### Create venv

```bash

python -m venv venv 
source venv/bin/activate


```

### Install requierements

```bash

pip install -r requirements.txt


```
